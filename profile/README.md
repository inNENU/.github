# inNENU

inNENU (曾用名 in东师) 系 Mr.Hope 个人公益项目，主要包含东师新生指南以及各类东师功能。

inNENU 于 2018 年 8 月最初上线，已服务 2018级、2019级、2020级、2021级、2022级、2023级六届新生。
